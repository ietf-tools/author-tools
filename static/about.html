About Author Tools

This service allows you to convert an Internet-Draft from one format into another, including rendered outputs. In the background this service uses id2xml, kramdown-rfc, mmark and xml2rfc, chaining them together as needed to deliver the requested conversion.

How to report bugs

  • For bugs and feature requests about IETF Author Tools website, create a new issue under Author Tools UI GitHub project.
  • For bugs and feature requests about IETF Author Tools API, create a new issue under Author Tools API GitHub project.
  • General discussions about IETF Author Tools and other IETF tools happens in Tools-discuss mailing list.

Allowed Domains

Following domains and their subdomains are allowed as URL inputs in Author Tools

  • ietf.org
  • rfc-editor.org
  • github.com
  • githubusercontent.com
  • github.io
  • gitlab.com
  • gitlab.io
  • codeberg.page
